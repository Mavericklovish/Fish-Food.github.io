Project Overview

Welcome to the Fish Feeding Simulation! In this interactive web experience, you'll dive into a virtual underwater world
teeming with colorful fishes. Your role is to provide nourishment to these aquatic inhabitants by dropping food into the water.
Watch as the fishes gracefully swim towards the food, eagerly gobbling it up.

  • How to Feed?

  • Click to Feed : Simply click anywhere on the screen to drop food into the water.
  • Watch the Fishes : Observe as the fishes detect the nearest food and swim towards it.
  • Enjoy the Spectacle : Sit back, relax, and enjoy sight of fishes eating food you dropped
    • in water to satisfy their hunger.

Objectives

  • Following were our objectives at each step when building this project :

  • To develop a user-friendly interface using HTML5 canvas.
  • Implement interactive functionality so that users can add food at the clicked location.
  • Create a visually appealing fish character that moves.
  • Design food items with varying appearances to add visual interest and realism to the scene.
  • Program the fish to detect the nearest food item and move towards it fluidly.
  • Document the codebase with clear comments and explanations to facilitate future maintenance and extension of the project.

  • Following are the learnings from this project :

  • Canvas drawing : Understanding how to use the HTML5 canvas element and drawing basic shapes.
  • Event handling : Implementing event listeners to detect user input, such as mouse clicks.
  • Animations and Transformations : Creating smooth animations using requestAnimationFrame and updating canvas elements dynamically.
  • Food eating : Implementing the logic for fish to detect nearest food and eat it.

Challenges

    Challenge 1 : Less Experience in JavaScript.


    • Situation : The project involves building a complex interactive and functional JavaScript program. However, lacking sufficient expertise in the language posed a challenge in initiating the project.
    • Task : To understand and implement various JavaScript concepts such as drawing shapes on Canvas, event handling and adding smooth transitions.
    • Action : Learned from online resources such as Youtube tutorials, documentation and help from friends.
    • Result : Successfully dividing tasks into smaller steps, learn concepts as we progressed using online resources and regularly practice.

    Challenge 2 : Implementing Event Listener


    • Situation : While developing the project, one of the tasks involves implementing functionality where food appears on the screen upon mouse-clicking events. This task required a good understanding of event handling.
    • Task : To implement an event listener for mouse clicks that triggers the appearance of food on the screen when the user interacts with the Canvas template. This involves understanding how to use JavaScript to detect mouse click events and dynamically manipulate the HTML and CSS to display the food item at the clicked location.
    • Action : Study the concept of event listeners in JavaScript, focusing on how they can be used to detect and respond to user interactions such as mouse clicks.
    • Result : Upon successful implementation of the event listener for mouse clicks, the project is exhibiting the desired behavior: whenever a user clicks anywhere on the screen, food will appear at the clicked location.

    Challenge 3 : Achieving smooth animation and design


    • Situation : Implementing smooth rendering and animation performance are crucial for delivering a good user experience. However, for achieving this performance on the canvas was challenging, due to limited experience in graphics programming and canvas coding.
    • Task : To add smooth rendering and animation on the HTML5 canvas. This involved understanding the factors that affect canvas performance and implementing strategies to optimize rendering and animation efficiency.
    • Action : Used efficient function drawing method such as a collectively calling all the functions in 'draw()'. Set up a requestAnimationFrame loop to control the animation frame rate and ensure smooth animation.
    • Result : Upon successful implementation of optimization techniques, the project is exhibiting smooth rendering and animation performance on the HTML5 canvas, providing users with an engaging and responsive visual experience.
